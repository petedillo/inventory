# My Express App

This is a simple Express application.

## Installation

To install the necessary dependencies, run the following command in the project directory:

```
npm install
```

## Running the Application

To start the application, use the following command:

```
npm start
```

The application will be running on `http://localhost:3000`.

## Project Structure

- `src/server.js`: Entry point of the application.
- `src/routes/index.js`: Defines the routes for the application.
- `package.json`: Configuration file for npm.
